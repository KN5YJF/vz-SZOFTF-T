## Beadás előtti ellenőrzőlista (értékelési szemponthoz igazítva)

- [x] WBS logikus és hierarchikus, minimum 4-6 fő feladattal.
- [x] Gantt diagram tartalmaz függőségeket, mérföldköveket, reális időbecslést.
- [x] Erőforrástervben humán és technikai erőforrás is szerepel.
- [x] Költségterv és NPV számítás konzisztens.
- [x] Kockázatokhoz kezelési terv és ellenőrzési pont tartozik.
- [x] Kommunikációs csatornák, riport gyakoriság és felelősök rögzítve vannak.
- [x] Nyelvhelyes, strukturált, beadásra kész formátum.
