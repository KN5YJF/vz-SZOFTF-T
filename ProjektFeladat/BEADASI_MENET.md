- [x] Projektterv elkészítve (WBS, Gantt, kockázat, költség, erőforrás, NPV)

Kimenet:

- Beadható projektterv-csomag (Projektterv + kiegészítő tervek).

## 3. Megvalósítási fázis

Állapot: beadható.

- [x] Dokumentációs sablon elkészítve
- [x] Funkciók implementálása
- [x] Funkcionális tesztkör futtatása (T1-T4) és hibák javítása
- [x] Tesztjegyzőkönyv végleges kitöltése
- [x] Megvalósítási dokumentáció véglegesítése

Megvalósítási lépések:

1. Angular projektstruktúra és UI váz véglegesítése. (kész)
2. Audio motor implementálása valós zongoramintákkal (Tone.js Sampler). (kész)
3. Egér és billentyűzet inputkezelés összekötése a hangkeltéssel. (kész)
4. Vizuális billentyű-visszajelzés és SCSS finomhangolás. (kész)
5. Funkcionális és kompatibilitási tesztek futtatása.
6. Hibajavítás, regressziós ellenőrzés, dokumentálás.

Kimenet:

- Működő böngészős zongora + tesztjegyzőkönyv + megvalósítási dokumentáció.
