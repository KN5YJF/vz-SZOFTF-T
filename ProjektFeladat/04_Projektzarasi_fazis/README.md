# 4. Projektzárási fázis

## Kötelező leadandó

- Záró prezentáció
- Projektet bemutató videó

## Lépések (beadható szint)

1. Összefoglalni a kiinduló célt és az elért eredményt.
2. Bemutatni a kész alkalmazást és a fő funkciókat.
3. Ismertetni a szerepeket és egyéni hozzájárulásokat.
4. Rövid reflexió: mi ment jól, mi volt nehéz, mi lenne a következő iteráció.
