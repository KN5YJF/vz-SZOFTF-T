# Virtuális zongora projekt

Szoftverfejlesztési projektfeladat egy böngészőben futó virtuális zongora alkalmazással és a hozzá tartozó projekt-dokumentációval.

## Tartalom

- `virtualis-zongora` - Angular alapú virtuális zongora alkalmazás
- `ProjektFeladat` - projektfázisok dokumentációi
- `Követelmény` - tantárgyi követelményanyag
- `BeadásraKész` - beadásra előkészített csomagok

## Alkalmazás indítása

```bash
cd virtualis-zongora
npm ci
npm start

Az alkalmazás eztkövetően érhető el: http://localhost:4200/

Build
bash

cd virtualis-zongora
npm run build


Projekt célja
A projekt célja egy egyszerű, böngészőben használható virtuális zongora elkészítése, valamint a fejlesztési folyamat dokumentálása projektalapító, tervezési, megvalósítási és zárási fázisok szerint.
